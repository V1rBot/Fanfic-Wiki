# Инструкция по запуску проекта

Этот проект состоит из двух частей: фронтенд (то, что вы видите в браузере) и бэкенд (сервер, который работает с файлами). Благодаря последним изменениям, для запуска всего проекта нужна всего одна команда.

## 1. Единоразовая настройка

Если вы этого еще не сделали, установите `nodemon`. Это утилита, которая автоматически перезапускает сервер при внесении изменений в код.

Откройте командную строку и выполните:
```bash
npm install -g nodemon
```

## 2. Запуск проекта

1.  Откройте командную строку (терминал).
2.  Перейдите в папку `server` в вашем проекте.
3.  Запустите сервер с помощью `nodemon`:
    ```bash
    nodemon server.js
    ```

После этого вы увидите сообщение `Backend server running at http://localhost:3103`.
**Больше ничего запускать не нужно!** Сервер одновременно обрабатывает и бэкенд-запросы, и отдает сам сайт.

## 3. Доступ к сайту

*   **На компьютере:** Откройте браузер и перейдите по адресу [http://localhost:3103](http://localhost:3103).
*   **На телефоне (в той же Wi-Fi сети):** Узнайте ваш локальный IP-адрес (с помощью команды `ipconfig` в командной строке) и откройте в браузере телефона `http://<ВАШ-IP-АДРЕС>:3103`.

---
_Эта инструкция отвечает на ваши вопросы о запуске фронтенда и использовании `nodemon`._

  ### 4. Как открыть сайт на телефоне не из локальной сети      


  Да, это возможно. Для этого используются так называемые       
  "туннелирующие" сервисы. Самый популярный и простой — ngrok.  

  Он создает временный публичный веб-адрес в интернете, который 
  перенаправляет все запросы на ваш локальный сервер.

  Как это сделать:


   1. Скачайте ngrok: Перейдите на официальный сайт ngrok.com  
      (https://ngrok.com/download), скачайте его и распакуйте  
      (установка не требуется). Возможно, потребуется
      зарегистрироваться.
   2. Запустите ваш проект, как описано в инструкции (в папке  
      server выполните nodemon server.js). Ваш локальный сервер
      должен работать на порту 3103.
   3. Запустите ngrok:
   4. Получите публичную ссылку: В окне ngrok вы увидите строку
      "Forwarding", а напротив нее — адрес, оканчивающийся на  
      .ngrok-free.app или .ngrok.io. Это и есть ваша публичная 
      ссылка.
  Теперь вы можете открыть эту ссылку на любом устройстве в       
  любой точке мира, и она будет показывать сайт, запущенный на    
  вашем домашнем компьютере (пока работают nodemon и ngrok).
